# Employee & Department Management System (MySQL)

## Project Description
This project is created using MySQL to manage employee, department,
and salary data. It demonstrates the use of joins, subqueries, group by,
and aggregate functions.

## Project Steps
1. Created database and tables (`departments`, `employees`, `salaries`)
2. Inserted sample data
3. Written 10 SQL queries to:
   - Display employees with their departments
   - Find employees with salary above 50,000
   - Department-wise employee count
   - Employees without salary assigned
   - Employee with highest salary
   - Average salary in IT department
   - Departments without employees
   - Employees joined after 2023
   - Total salary by department
   - Employees earning above department average

## Technologies Used
- MySQL 8.0
